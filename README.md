# Grafos em Java – Multigrafo com Arestas Nomeadas

Este projeto implementa um **multigrafo não direcionado** em Java, com arestas nomeadas.  
O grafo permite múltiplas arestas entre os mesmos vértices e possibilita armazenar e exibir conexões de forma organizada.
O grafo se baseou no estudo sobre as linhas de ônibus DP de Diadema, analisando os bairros da cidade por quais passam.

---

## Estrutura do Projeto

- `Multigrafo.java` – Classe que representa o grafo, com métodos para adicionar vértices e arestas, e imprimir o grafo.  
- `Aresta.java` – Classe que representa uma aresta, com nome e vértices conectados.  
- `Main.java` – Exemplo de uso do grafo: criação de vértices, adição de arestas e impressão do grafo.
